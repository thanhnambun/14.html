oán tử bằng (==): So sánh giá trị của hai biểu thức. Nếu giá trị của hai biểu thức bằng nhau, kết quả là true. Ví dụ: 2 == '2' sẽ trả về true. Toán tử bằng tuyệt đối (===): So sánh giá trị và kiểu dữ liệu của hai biểu thức. Chỉ khi giá trị và kiểu dữ liệu của hai biểu thức đều giống nhau, kết quả là true. Ví dụ: 2 === '2' sẽ trả về false. Toán tử khác (!= hoặc !==): So sánh giá trị của hai biểu thức. Nếu giá trị của hai biểu thức khác nhau, kết quả là true. Toán tử != so sánh giá trị, còn toán tử !== so sánh giá trị và kiểu dữ liệu. Ví dụ: 2 != '2' sẽ trả về false, trong khi 2 !== '2' sẽ trả về true. Toán tử lớn hơn (>): So sánh hai giá trị và trả về true nếu giá trị bên trái lớn hơn giá trị bên phải. Ví dụ: 5 > 3 sẽ trả về true. Toán tử nhỏ hơn (<): So sánh hai giá trị và trả về true nếu giá trị bên trái nhỏ hơn giá trị bên phải. Ví dụ: 2 < 8 sẽ trả về true. Toán tử lớn hơn hoặc bằng (>=): So sánh hai giá trị và trả về true nếu giá trị bên trái lớn hơn hoặc bằng giá trị bên phải. Ví dụ: 4 >= 4 sẽ trả về true. Toán tử nhỏ hơn hoặc bằng (<=): So sánh hai giá trị và trả về true nếu giá trị bên trái nhỏ hơn hoặc bằng giá trị bên phải. Ví dụ: 3 <= 6 sẽ trả về true.
